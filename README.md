### Installation

Challenge Tornado requires [Python](https://www.python.org/) v3.6+ and [Pip](https://pypi.org/project/pip/) to run

Clone repository

```sh
$ git clone ...
```

Go directory

```sh
$ cd challenge-tornado
```

Install the dependencies

```sh
$ pip3 install -r requirements/local.txt
```

To run tests

```sh
$ pytest
```

Run service
```sh
$ python3 main.py
```

Go to the app
```sh
http://localhost:8888/
```
